# Sistema de Información (SIA) - Gestión Inmobiliaria

**Asignatura:** INF2236-1 - Programación Avanzada 
**Periodo:** 2026-1
**Institución:** Pontificia Universidad Católica de Valparaíso 

## Descripción del Proyecto
Este proyecto es un Sistema de Información (SIA) desarrollado en Java para gestionar ventas de departamentos dentro de distintos proyectos inmobiliarios. El sistema administra un conjunto de proyectos que ingresan al sistema y maneja la fluctuación dinámica de la demanda y los precios (mayor demanda implica mayores precios).

## Características Principales
* Gestión de Proyectos Inmobiliarios (Elementos Tipo A).
* Gestión de Departamentos (Elementos Tipo B).
* Cálculo de precios basado en niveles de demanda.
* Interfaz dual: Consola y Ventanas (GUI).
* Persistencia de datos (Carga inicial y guardado al salir).

## Tecnologías Utilizadas
* **Lenguaje:** Java (JDK 8)
* **IDE:** Eclipse IDE for Java Developers

## Instrucciones de Instalación y Ejecución
1. Clonar este repositorio en la máquina local utilizando el comando: 
   `git clone https://github.com/FrancoAllendes/SIA_Inmobiliaria.git`
2. Abrir Eclipse IDE.
3. Ir a `File` > `Import` > `General` > `Projects from Folder or Archive`.
4. Seleccionar el directorio clonado y finalizar la importación.
5. Asegurarse de tener configurado el entorno de ejecución JavaSE-1.8.
6. Ejecutar la clase principal (`SistemaInmobiliaria.java`) como una aplicación Java (`Run As` > `Java Application`).
